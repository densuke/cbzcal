# Suggested Commands

## Development
- `cargo build`: Build the project.
- `cargo run -- <args>`: Run the CLI tool.
- `cargo test`: Run tests.
- `cargo clippy`: Run linting.
- `cargo fmt`: Format code.

## CLI Usage Examples
- `cargo run -- doctor`: Configuration check.
- `cargo run -- probe-login`: Login test for Cybozu HTML backend.
- `cargo run -- events list`: List events.
- `cargo run -- events --prompt "tomorrow at 3pm meeting"`: Interactive command generation via LLM.
